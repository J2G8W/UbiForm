◆ searchForResourceDiscoveryHubs()

void ResourceDiscoveryConnEndpoint::searchForResourceDiscoveryHubs ( )

We search for resource discovery hubs by looking for components with background listeners. We then try to connect to the given RDHs, when we find 1 we stop as this is a slowwww process. We look for background listeners as each component has a background listener but there may just be one RDH on the network, so we are more likely to get a positive result
